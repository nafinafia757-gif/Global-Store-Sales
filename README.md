# Global-Store-Sales-Dashboard-PowerBI

##Project Overview:

This project presents an interactive sales dashboard for a global store with branches across different countries, regions, and markets. The dashboard helps stakeholders understand sales performance, shipping patterns, and geographical distribution of sales.

##Dataset:

The dataset contains sales transaction details including: Order ID, Order Date, Ship Mode, Sales, Orders, City, State, Region, Country, Market.

##Data Cleaning:

Basic data cleaning was done using Power Query in Power BI:

Removed null and duplicate values. Corrected data types. Renamed columns. Standardized location names. Checked ship mode consistency.

##Slicers:
The dashboard includes slicers to filter data by: 
Country

Market

Region.. These filters make the report interactive for users.

Bar charts showing total sales by Market and Region.

##Tables: 

Each chart has a related table to display the exact numerical values for better understanding.

##Insights: 
Identified top performing states and cities.

Understood which ship mode is most used.

Analyzed sales performance across regions and markets.

Observed global sales distribution using the map.

##How to Use:

Use the slicers (Country, Market, Region) to filter data.

Hover on visuals to see detailed values.

Refer tables for exact numbers. Use the map to understand geographical sales.

##Conclusion:

This dashboard turns raw sales data into clear insights using PowerBI.

Stakeholderscan easily analyze sales by location,market,region, and shipmode to support better business decisions.
